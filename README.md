# Hello Agent

AI Agent 项目集合，包含多个独立的 AI 应用项目。

## 项目列表

| 项目 | 状态 | 说明 |
|------|------|------|
| [ai_chef](./ai_chef/) | 🚧 开发中 | 会话式 AI 私厨助手 |
| [photo-score](./photo-score/) | 🚧 开发中 | 摄影图片点评系统 |

## 项目管理

每个子项目遵循统一结构：

```
project-name/
├── README.md          # 项目说明
├── TASK.md           # 任务清单
├── requirements.txt/pyproject.toml  # 依赖
└── src/              # 源码
```

## 快速开始

```bash
# 进入具体项目
cd photo-score

# 查看项目任务
cat TASK.md

# 安装依赖
pip install -r requirements.txt
```
